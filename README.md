A simple addressList application.

## Get Started

Install deps first:

```
pnpm i
```

### Dev

Run commands below. 

```
pnpm run dev // start dev server

pnpm run start:api // start backend service
```

Open application in browser and login as:

```
username: jeanmay
password: password123456
```


### Test

Run unit tests:

```
pnpm run test:unit
```

Run e2e tests:

```
pnpm run test:e2e
```
